# Gaming Analytics Database

## 📌 Overview

A relational database project designed to analyze player behavior, gameplay performance, and monetization patterns in an online game using **synthetic data**. Combines **SQL** and **Python** to generate data-driven insights.

---

## Objectives

* Design a **normalized database schema**
* Simulate realistic gameplay data
* Analyze player activity, performance, and spending
* Extract insights using SQL queries and visualizations

---

## Key Components

* **Core Entities:** Player, GameSession, Level, Mission, Item, Achievement, Friendship, Event
* Fully normalized schema with **PK/FK constraints**
* Supports behavioral, social, and financial analysis

---

## Methodology

* Synthetic dataset generation (privacy-safe & consistent)
* Database implementation in **PostgreSQL**
* Analysis via **SQL queries** + **Python visualizations**

---

## Key Insights

* Player behavior varies significantly across users
* Certain levels/missions act as **difficulty bottlenecks**
* Revenue is concentrated in a few items
* Weak positive relationship between **playtime and spending**
* Social and event participation influence engagement

---

## 🛠️ Tech Stack

* PostgreSQL
* SQL
* Python
